Synchronisation Google Sheets

Guide de configuration
1
Créer un Google Sheet partagé
Ouvrir sheets.google.com, créer un classeur, puis Fichier → Partager → Tout le monde avec le lien peut modifier.
2
Déployer le script Apps Script
Dans le Sheet : Extensions → Apps Script. Coller ce code :
function doPost(e){ var ss=SpreadsheetApp.getActiveSpreadsheet(); var data=JSON.parse(e.postData.contents); var sheet=ss.getSheetByName(data.dossier)||ss.insertSheet(data.dossier); if(data.action==='write'){ sheet.clearContents(); sheet.appendRow(['date','piece','journal','libelle','compte','libelle_ligne','debit','credit','tiers']); data.rows.forEach(function(r){sheet.appendRow(r);}); return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } if(data.action==='read'){ var vals=sheet.getDataRange().getValues(); return ContentService.createTextOutput(JSON.stringify({ok:true,rows:vals})).setMimeType(ContentService.MimeType.JSON); } } function doGet(e){return doPost(e);}
Cliquer Déployer → Nouveau déploiement → Application Web. Exécuter : Moi. Accès : Tout le monde. Copier l'URL.
3
Coller l'URL ci-dessous
4
Autres postes
Sur chaque autre PC : ouvrir ComptaSYSCO.html → Google Sheets Sync → coller la même URL → Enregistrer → Récupérer depuis Sheets.
